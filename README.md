# Basic to-do React App

Inspired by Dona.ai to-do list https://dona.ai/

## Description

To-do list app made with react library and create-react-app.

## Tools and libraries used

- fontawesome icons
- Inter google font
- nanoid

## React features used

- Class components
- Function components
- Stateless function components
- Controlled components

## Other features

- Pagination
- Conditional styling
